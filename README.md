# 64-tap-16-bit-Finite-Impulse-Response-Filter
# Maria Eleni Batatoudi, mb5017
# CSEE W4823, Advanced Logic Design, Columbia University
This project presents the design and implementation of a 64-tap, 16-bit Finite Impulse Response (FIR) filter using Verilog. The design integrates a dual-clock FIFO, coefficient memory, an arithmetic logic unit, and registers to ensure efficient data flow and processing.
